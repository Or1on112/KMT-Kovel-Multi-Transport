# KMT — Kovel Multi Transport

Welcome to the development repository for the Kovel Multi Transport (KMT) website.

## Getting Started
The project is organized into a modular structure to ensure scalability and clarity. Detailed instructions for AI assistants can be found in the `agent/instructions/` directory.

## Directory Structure
- `/agent`: Instructions and guides for AI collaboration.
- `/assets`: Images, fonts, and global styles.
- `/content`: Copy and structural content.
- `/docs`: Project logs and technical documentation.
- `/references`: Design inspiration and reference materials.
