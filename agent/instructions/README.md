# Agent Instructions - Kovel Porto Project

## Project Overview

This project focuses on building a premium, high-fidelity website. The goal is to create a visually stunning experience using modern web technologies.

## Technology Stack

- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: Vanilla CSS (Modern CSS features like Variables, Flexbox, Grid).
- **Aesthetics**: High emphasis on premium design, glassmorphism, smooth animations, and curated color palettes.

## Project Structure

- `assets/images/`: Store all project-related images and graphics.
- `assets/fonts/`: Custom typography files.
- `assets/styles/`: Global CSS tokens, themes, and utility styles.
- `agent/instructions/DESIGN_SYSTEM.md`: Specific design rules and aesthetics for "Industrial Rationalism".
- `agent/instructions/`: Detailed guides and context for AI models working on this project.
- `references/`: Screenshots, design inspirations, and documents for visual direction.
- `content/`: Text content, copy, and localization data.
- `docs/`: Technical documentation and project planning.

## Collaboration Rules

1. **Language**:
   - The user communicates in **Russian**.
   - All internal instructions, code comments, file names, and technical documentation must be in **English**.
2. **Design First**: Always prioritize visual excellence. Before writing complex logic, ensure the UI structure and styling feel "premium".
3. **No Placeholders**: Use generated images or real assets. Avoid empty `div`s or "lorem ipsum" where possible during the final stages.
4. **Iterative Development**: Build components in isolation, then integrate them into the main layout.
