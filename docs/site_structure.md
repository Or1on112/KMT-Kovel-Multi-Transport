## 2. Сторінки та блоки

### Головна сторінка

1. **Hero Section**
   - **Заголовок:** Комплексні рішення. Економія часу та ресурсів.
   - **Підзаголовок:** Митно-логістичний вузол. Ковель. Міст між Україною та ЄС.
   - **CTA:** [Отримати консультацію]
2. **Розташування (Features Grid)**
   - Траса Київ — Ягодин (М07)
   - 60 км до пункту Ягодин — Дорогуськ
   - Доступ до пунктів Устилуг та Ізов
3. **Огляд послуг (Services Summary)**
   - Митне оформлення
   - Складське зберігання
   - Брокерський супровід
   - Інфраструктура для вантажівок
4. **Переваги (Advantages List)**
   - Стратегічна локація
   - Власний митний пост та Держспоживслужба
   - Інтермодальність: колії 1520 мм та 1435 мм
   - Контейнерний термінал
   - Контроль та безпека (WMS)

### Про термінал

1. **Компанія:** Сполучна ланка між Україною та ЄС. Прямий доступ до ринків.
2. **Логістика:** Робота з двома типами колій (1435/1520 мм). Швидкий транзит.
3. **Єдине вікно:** Митниця та ветеринарно-фітосанітарний контроль на одній локації.
4. **Місія:** Оптимізація оформлення. Цілісність вантажів.

### Послуги (Детально)

1. **Митне оформлення**
   - *Опис:* Брокерський супровід. Система «Єдине вікно».
   - **CTA:** [Розрахувати вартість] -> `form_customs_clearance`
2. **Паркінг**
   - *Опис:* Охоронювана зона очікування. Відеонагляд.
   - **CTA:** [Забронювати місце] -> `form_parking`
3. **Митний склад**
   - *Опис:* Закритий склад (345 м²), відкритий майданчик (6000 м²). Система WMS.
   - **CTA:** [Замовити склад] -> `form_general_request`
4. **Залізниця**
   - *Опис:* Прийом вагонів. Перевантаження: вагон-вагон, авто-вагон. Зміна колії.
   - **CTA:** [Замовити перевантаження] -> `form_railway`
5. **Контейнерний майданчик**
   - *Опис:* Обробка, зберігання, стафірування контейнерів.
   - **CTA:** [Замовити обробку] -> `form_general_request`

### Інфраструктура

- **Митна:** Пост, вет/фіто контроль.
- **Складська:** Криті та відкриті площі.
- **Транспортна:** Прямий заїзд, маневрові майданчики.
- **Залізнична:** Комбінована колія.
- **Безпека:** 24/7 моніторинг.

### Переваги (Matrix)

- **Оперативність:** Мінімальні терміни процедур.
- **Економія:** Пряме зниження логістичних витрат.
- **Гнучкість:** Обробка всіх типів вантажів.
- **Контроль:** Повний цифровий облік (WMS).
- **Надійність:** Физична та юридична безпека.

### Інфо-центр

1. **Процес оформлення авто:** Прибуття -> Документи -> Митниця -> Склад -> Випуск.
2. **Як стати клієнтом:** Консультація -> Підготовка -> Договір.

## 3. Форми та захоплення лідів

### Форма: Митне оформлення (`form_customs_clearance`)

- **Контакти:** Назва компанії, Особа, Телефон, Email.
- **Операція:** Імпорт / Експорт / Транзит.
- **Вантаж:** Назва, Код УКТЗЕД, Походження, Вага, Вартість.
- **Транспорт:** Авто / Залізниця / Контейнер.
- **Додатки:** Склад, Перевантаження, Контроль (вет/фіто).
- **Файли:** Завантаження документів (Інвойс, CMR, Пакувальний).

### Форма: Паркінг (`form_parking`)

- **Контакти:** Компанія, Телефон.
- **Транспорт:** Тип, Держномер авто та причепу.
- **Причина:** Імпорт / Експорт / Транзит / Розвантаження.
- **Термін:** До 24 год / Більше 24 год.
- **Прибуття:** Дата та час.
- **Додатково:** Послуги завантаження/розвантаження.

### Форма: Залізниця (`form_railway`)

- **Контакти:** Компанія, Телефон.
- **Маршрут:** Відправлення -> Призначення (станції).
- **Тип робіт:** Вагон-вагон / Вагон-авто / Контейнери.
- **Колія:** Перехід з 1435 на 1520 або навпаки.
- **Рухомий склад:** Зерновози, криті, платформи.
- **Вантаж:** Назва, Вага, Кількість одиниць.
- **Дата:** Очікуване прибуття.

### Форма: Загальна заявка (`form_general_request`)
- **Контакти:** Компанія, Телефон, Email.
- **Тема:** Митний склад / Контейнерний майданчик / Інше.
- **Опис:** Короткий опис потреби.
- **Об'єм:** Орієнтовна вага або кількість одиниць.
